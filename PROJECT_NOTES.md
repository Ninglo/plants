## 5. MVP 功能范围（今晚）

### 首页 / 登录页
- 学校 logo + 标题
- 账号 + 密码输入
- 登录按钮（调用官网 API）
- "暂时跳过" 链接
- 登录后：显示 Welcome + slogan + 右上角当前周数 + 班级列表

### 主流程（核心）
1. 选择发放模块（勾选）：基础落实（必选）/ 每日开口 / 课堂参与 / 个性化奖励
2. 上传对应文件
3. 系统汇总数据 → 预览表格（每人明细）
4. 老师手动微调（课堂参与批量修改）
5. 生成 → 输出图片 + Excel

## 6. 数据源文件格式

### 基础落实：`{班级号}_学生个人数据_{timestamp}.xlsx`
- Sheet **数据底表**：行0=大组(Week/Unit), 行1=资源名称(含[类型]), 行2=附属列名, 行3+= 学生数据
- 学生标识：**学号**（唯一），班级号从**文件名**提取（正则 `[A-Z]\d{3}`）
- 排序：按英文名字母序

### 每日开口：`打卡情况{timestamp}.xlsx`
- Sheet **data**：学号 / 英文名 / 中文名 / 日期列(1/空) / 总计
- 与基础落实用**学号**匹配

### 目标上传格式：`目标excel格式.xlsx`
- Sheet **学生列表**：英文名 / 中文名 / 学号 / 发放MP数量

---

## 7. 资源类型 & 附属列

| 类型 | 附属列 | 数据格式 |
|------|--------|----------|
| `[音频]` | 是否完成 / 播放进度 / 播放时长 | 文字 / 字符串% / 整数秒 |
| `[主题表达积累]` | 是否完成 / 是否达到词王 / 词王平均正确率 / 重复环节平均正确率 | 文字 / 有值=是 / 字符串% |
| `[小挑战]` | 是否完成（得分列全为空） | 文字 |
| `[AI语音]` | 是否完成 / 平均分 / 第一次得分 | 文字 / 字符串% |
| `[测试]` | 是否完成 / 分数(15) | 文字 / 整数满分15 |
| `[复合资源]` | 是否完成 / +1值 | 文字 / 整数(系统预算) |

> 全局规则：`进行中` = 未完成（0分）

## 9. 四大发放模块

| 模块 | 是否必选 | 数据来源 | 特殊逻辑 |
|------|----------|----------|----------|
| 基础落实 | 必选 | 学生个人数据 xlsx | 学号 = 主键；班级号从文件名提取 |
| 每日开口 | 可选 | 打卡情况 xlsx | 用学号匹配，取总计列 |
| 课堂参与 | 可选 | 老师手动 | 默认值(0.2/0.3)+多选批量改值 |
| 个性化奖励 | 可选（选作） | 老师手动 | 自由新增任务+指定学生+金额 |
